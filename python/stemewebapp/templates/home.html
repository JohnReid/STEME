Welcome to the STEME motif finder

STEME is an accurate efficient motif finder based on the MEME algorithm. We have made an installation of STEME available on this web site. Please feel free to submit your jobs to this web site. If you want more control over STEME's options or you wish to run STEME on your own servers please download and install the STEME software.
